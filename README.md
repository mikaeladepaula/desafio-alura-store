## 📝 Descrição do Projeto

Este projeto foi desenvolvido como parte do Alura Store Challenge. O objetivo principal é auxiliar o Senhor João, proprietário da rede **Alura Store**, a identificar qual de suas quatro unidades possui a menor eficiência operacional para ser vendida, viabilizando assim um novo empreendimento.

A análise utiliza técnicas de Ciência de Dados para processar informações de faturamento, logística e satisfação do cliente, transformando dados brutos em decisões estratégicas fundamentadas.

## ✅ Funcionalidades (Análises Realizadas)

* **Análise de Faturamento**: Cálculo do faturamento total por unidade para identificar a performance financeira de cada loja.
* **Vendas por Categoria**: Mapeamento das categorias mais populares para entender o perfil de consumo de cada região.
* **Média de Avaliação**: Mensuração da satisfação dos clientes através das notas de avaliação da compra.
* **Desempenho de Produtos**: Identificação dos itens "queridinhos" (mais vendidos) e dos produtos com baixa saída em cada unidade.
* **Custo Logístico**: Análise do frete médio pago pelos clientes para entender o impacto da logística nas vendas.
